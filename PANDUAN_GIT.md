# 📘 Panduan Cepat Pengembangan CV Portfolio

Gunakan referensi ini saat Anda mengerjakan proyek.

## 🚀 Menjalankan Server Lokal
```powershell
npm run dev
```
Akses di: [http://localhost:3000](http://localhost:3000)

---

## 💾 Menyimpan ke GitHub (Workflow Git)
Lakukan 3 perintah ini secara berurutan:

1. **Pilih file yang diubah:**
   ```powershell
   git add .
   ```
2. **Kunci perubahan dengan pesan:**
   ```powershell
   git commit -m "Tulis apa yang Anda ubah"
   ```
3. **Kirim ke internet (GitHub):**
   ```powershell
   git push origin main
   ```

---

## 🛠️ Tips Cepat
- **Hentikan Server Lokal:** Tekan `Ctrl + C` di terminal.
- **Cek Status:** Selalu gunakan `git status` untuk melihat file yang "tercecer".
